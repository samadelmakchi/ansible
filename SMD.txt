>> Log
▫️Fluentd       ➡️ aggregator/log processor برای ورودی‌ها (از Podها، اپلیکیشن‌ها، یا حتی Beats)
▫️Elasticsearch ➡️ ذخیره و جستجو لاگ‌ها
▫️Kibana        ➡️ نمایش، فیلتر و مانیتورینگ لاگ‌ها
▫️Beats
   ▫️Filebeat   ➡️ جمع‌آوری لاگ‌های nginx، PHP (Apache/nginx logs، PHP error logs)
   ▫️Metricbeat ➡️ جمع‌آوری متریک‌های سیستم و MySQL (CPU، RAM، query load و...)
   ▫️Packetbeat ➡️ ضبط و تحلیل ترافیک لایه شبکه و اپلیکیشن (HTTP, MySQL, DNS, ... )
   ▫️Heartbeat  ➡️ Ping و بررسی وضعیت سلامت سرویس‌ها (uptime monitoring)
   ▫️Auditbeat  ➡️ بررسی امنیتی سیستم: تغییرات فایل، یوزرها، دسترسی‌ها
▫️Logstash      ❌ وقتی پردازش سنگین‌تری لازمه
